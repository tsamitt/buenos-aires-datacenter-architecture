# Technologies Used

Nutanix AHV

Nutanix AHV is a hypervisor used to virtualize servers and run multiple virtual machines on a shared physical infrastructure.

F5 Load Balancers

F5 load balancers distribute application traffic across multiple backend servers to improve availability and performance.

Spine–Leaf Network Architecture

A spine-leaf architecture provides predictable low latency and high bandwidth communication between servers.

Enterprise Firewalls

Firewalls provide network segmentation and enforce security policies.

Edge Routers

Edge routers connect the data center to external networks and enable communication with remote locations.
